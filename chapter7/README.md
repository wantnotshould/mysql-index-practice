# 第 7 章

索引，在 MySQL 中也叫作键（key），是存储引擎用于快速找到记录的一种数据结构

**使用索引的优点**

- 索引大大减少了服务器需要扫描的数据量
- 索引可以帮助服务器避免排序和*临时表（GROUP BY, DISTINCT, JOIN）*
- 索引可以将随机I/O变为顺序I/O
